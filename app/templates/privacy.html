About {{bot_name}}

Anonymous Session Cookie

The chatbot uses an anonymous session cookie necessary for it to function, and for the core research purposes of the original study for which it was created. The cookie lasts for the duration of a browsing session. Here are more details about it:

Cookie Name:
anon_session
Expires:
Session
Content:
Anonymous session identifier
Purpose:
Used by the server to maintain an anonymised user session, to enable the chatbot to maintain a temporally consistent conversation with the user, and to enable the collection of anonymous data essential for the core research purposes of the original study for which it was created.

Data Collected

The following data is collected from each chat session:

  • An anonymous session identifier that uniquely and anonymously identifies the session
  • When the chat session started (date and time)
  • When the chat session ended (date and time)
  • Record of the user's agreement to {User agreement part 1} and {User agreement part 2}

For each chat message sent in the session, the following data is collected:

  • Anonymous session identifier (so that each message can be linked to its corresponding session)
  • Content of the message sent by the user or chatbot
  • When the message was sent by the user or chatbot (date and time)
